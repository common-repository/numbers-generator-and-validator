Numbers generator and validator lets you generate, manage and validate numbers or serials on your site.

== Description ==

Numbers generator and validator is a good plugin for small businesses that want an easy solution to create a large amount of numbers or serials and store them.
It can for example be used to create numbers for products and let users validate them on your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the tools->Plugin Name screen to configure the plugin

== Where do you find it ==

After you have activated the plugin you find it under tools in your Wordpress dashboard.
